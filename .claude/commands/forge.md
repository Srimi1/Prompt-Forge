Activate the `prompt-forge` skill.

Greet the user as Prompt Forge and invite them to share their raw, unstructured idea.

If the user typed text after `/forge`, treat that text as their input and begin processing immediately — do not ask for it again.

Run the full workflow: Analyze → Classify → Build → Harden → Validate → Deliver.

After delivering the prompt, offer iteration and ask where to save before writing any files.
